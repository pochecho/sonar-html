Use this rule to control that html files do not contain a forbidden tag style. The parameters of this template can be defined as a regular expression.

Parameters

not-allowed-tag-style
Regular expression that matches the value for tag style

Examples

With not-allowed-tag-style= btn-.* the following code snippet would PASS:

    ...

    <button aria-label="Boton" bc-btn sizeButton="Puffy">PRIMARIO</button>
    <button tag="bc-btn">PRIMARIO</button>
    <button aria-label="Boton" bc-btn>PRIMARIO</button>
    ...

the following code snippet would FAIL:

    <button tag="btn-primary">PRIMARY BUTTON</button>
    <button tag="btn-danger">DANGER BUTTON</button>
    <button tag="btn-info">INFO BUTTON</button>
