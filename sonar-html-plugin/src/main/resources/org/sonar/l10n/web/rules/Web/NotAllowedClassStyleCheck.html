Use this rule to control that html files do not contain a forbidden class style. The parameters of this template can be defined as a regular expression.

Parameters

not-allowed-class-style
Regular expression that matches the value for class style

Examples

With not-allowed-class-style= btn-.* the following code snippet would PASS:

    ...

    <button aria-label="Boton" bc-btn sizeButton="Puffy">PRIMARIO</button>
    <button class="bc-btn">PRIMARIO</button>
    <button aria-label="Boton" bc-btn>PRIMARIO</button>
    ...

the following code snippet would FAIL:

    <button class="btn-primary">PRIMARY BUTTON</button>
    <button class="btn-danger">DANGER BUTTON</button>
    <button class="btn-info">INFO BUTTON</button>
